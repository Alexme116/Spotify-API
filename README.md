# Spotify API

## Execution Instructions

1. **Clone this repository**
    ```bash
    git clone https://github.com/Alexme116/Spotify-API.git
    ```

2. **Change to cloned project directory**
    ```bash
    cd Spotify-API
    ```

3. **Install packages and dependencies**
    ```bash
    npm install
    ```

4. **Run the localhost**
    ```bash
    npm run dev
    ```
